# LenderoHUB Playground — Training Exercises

Ejercicios progresivos para aprender desarrollo full-stack (Express + Next.js + MongoDB) usando herramientas AI (Claude Code + Cursor).

## Estructura

| Level | Tema | Ejercicios | Horas estimadas |
|-------|------|-----------|-----------------|
| 0 | Foundations | 4 | 6-8h |
| 1 | Frontend Basics | 4 | 12-16h |
| 2 | Backend Basics | 4 | 12-16h |
| 3 | Full-Stack Features | 3 | 15-20h |
| 4 | Advanced Patterns | 3 | 20-25h |
| 5 | Capstone | 1 | 15-20h |
| **Total** | | **19** | **~80-105h** |

## Como empezar

1. Completa el setup en `level-0/01-environment-setup.md`
2. Sigue los ejercicios en orden dentro de cada nivel
3. Marca tu progreso en `PROGRESS.md`
4. Lee `AI-WORKFLOW.md` para la guia de cuando usar cada herramienta AI

## Reglas

- **Completa cada ejercicio antes de avanzar al siguiente** dentro del mismo nivel
- **Levels 0-2 son secuenciales** (cada uno construye sobre el anterior)
- **Levels 3-5 pueden hacerse en cualquier orden** dentro del nivel
- **Siempre lee "Archivos de Referencia"** antes de empezar un ejercicio
- **Usa herramientas AI** — el objetivo es aprender a usarlas efectivamente, no memorizarlas

## Evaluacion

Cada ejercicio tiene "Criterios de Aceptacion" — una checklist que DEBE cumplirse al 100%. Si no puedes marcar todos los checkboxes, el ejercicio no esta completo.
